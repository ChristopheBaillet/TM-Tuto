What is this site all about ?

This site is designed to be a bridge between trackmania players wanting to better themselves and the documentation containing some of the knowledge you need to know. Aswell as community based coaching on the Evo discord.

Enjoy your stay on this website and I hope you will learn something that will make you go faster in the game.

Styles

Tech

Tech is a technical style that requires you to know how to drift and while driving on road blocks, maintaining as much speed as you can going through the turns

I want to learn Tech
Dirt

Dirt is a technical style like tech on road in which you need to be able to drive on different dirt blocks while maintaining the most speed you can during, and after the turn

I want to learn Dirt
Fullspeed

Fullspeed is a style which consists of never letting go of the accelerate button and going really fast through wallrides, loopings, turn-overs, and long platform/road/dirt/grass.
